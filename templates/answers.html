For task 1.1 I added the recent tweets for each airline, as I felt that the tweets in combination with the airline sentiment pie charts, provided a visualization that informed each other. The pie charts show the ratio of positive, negative and neutral interactions with respective airlines, and the recent tweets provide some context behind those interactions. I believe this creates an impactful visualization for users, as it could help users to determine which airline they might decide to use for their flight. I also added two buttons, which toggle the display on and off for the recent tweets, I believe this helps to make the screen more streamlined and easier to use on desktop and mobile devices. I also used flexbox, so that the pie charts and recent tweets text display would adjust nicely to the screen size.
